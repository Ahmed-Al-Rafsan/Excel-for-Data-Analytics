# Excel for Data Analytics – Day 7: Data Cleaning



**Objective**

The goal of Day 7 was to clean messy, real-world Excel data and make it analysis-ready using simple and effective techniques.



This day focused on \*\*practical judgment\*\*, not overengineering.



---



**Topics Covered**

\- Removing unwanted spaces using `TRIM()`

\- Fixing internal spacing issues using `SUBSTITUTE()`

\- Correcting known data errors using \*\*Find \& Replace\*\*

\- Understanding when to use formulas vs manual cleaning

\- Preparing clean data for PivotTables, Charts, and BI tools



---



**Key Learning**

\- One formula does not solve every problem

\- The simplest correct solution is often the best

\- Data cleaning is about \*\*choosing the right tool\*\*, not memorizing functions

\- Excel and SQL handle data cleaning differently



---



**Outcome**

\- Cleaned and standardized Customer IDs

\- Removed hidden and internal spaces

\- Dataset is now safe for analysis, charts, and dashboards
